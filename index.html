Variables: we define our variables with var and followed by an identifier (the name we will give to it) this should be unique like in the example:
var x = 6;
Other way to create a variable is:
var myNewVar;
and the we can assign the value:
myNewVar = 'Ixshel';

JavaScript data types, in JavaScript types are dynamic but is a good practice to tell what kind of data type are you holding on a variable:

  • Strings: 'Ixshel' or "Ixshel"
  • Numbers: 10 or 10.00
  • Booleans: true or false
  • Arrays: var myArray = [ 1, "ixshel", { school: 'Ipn' }, true ]
  • Objects: var person = { firstName: 'Ixshel', lastName: 'Escamilla' }
  • Undefined
  • Null

Also even we didnt write the code we have a way to know what type of data is it, we can achieve this with typeof Operator typeof "Ixshel" or typeof 3.1416

What happens if a variable doesnt have a value? then we say is undefined this is a type itself.

What happens if a variable is assigned with nothing? then we say is null this is an Object.

Scope: this is the range of a variable, this can be global or local;

  • Global: you can access from any where of the code
  • Local: you can access to it only from the function that contains it

example 1: 
                
                        // Global
                        var me = 'Ixshel';
                        function sayHi() {
                            alert('Hi! ' + me);
                        }
        
                        // Local
                        function hiBack() {
                            var you = 'Itzel';
                            alert('Hi there! ' + you);
                        }
        
                        sayHi();
                        hiBack();
        
                        console.log(me); // will give 'Ixshel';
                        console.log(you); // will throw an error;
